PRE-IMPLEMENTACIÓN
═══════════════════════════════════════════════════════════════════════════════

Hardware:
  □ Arduino Uno disponible
  □ Ethernet Shield disponible
  □ SD Card (FAT16) disponible
  □ Cable Ethernet disponible
  □ Resistencias 30kΩ y 10kΩ para divisores (3 juegos)
  □ 3 LEDs (Rojo, Amarillo, Verde)
  □ 4 LEDs de control (D6-D9)
  □ LDR sensor de luz disponible
  □ Resistencia pull-up para LDR (10kΩ)
  □ Multímetro para calibración

Software:
  □ Arduino IDE instalado
  □ Librerías: SPI, Ethernet, SD (incluidas en IDE)
  □ Navegador web moderno
  □ Editor de texto para modificar HTML (opcional)

CONFIGURACIÓN DE RED
═══════════════════════════════════════════════════════════════════════════════

Red Disponible:

MONTAJE CIRCUITO
═══════════════════════════════════════════════════════════════════════════════

Divisores de Voltaje (para A2, A3, A4):
  □ R1 (30kΩ) desde voltaje positivo hasta A_PIN
  □ R2 (10kΩ) desde A_PIN hasta GND
  □ Capacitor 100nF entre A_PIN y GND (opcional pero recomendado)
  □ REPETIR PARA LOS 3 PINES (A2, A3, A4)

Sensor LDR (A5):
  □ Resistencia 10kΩ desde +5V hasta A5
  □ LDR desde A5 hasta GND
  □ Sensor se oscurece cuando baja luz

LEDs de Estado (D2, D3, D5):
  □ LED Rojo en D2 (ánodo a D2 a través de 220Ω)
  □ LED Amarillo en D3 (ánodo a D3 a través de 220Ω)
  □ LED Verde en D5 (ánodo a D5 a través de 220Ω)
  □ Todos los cátodos a GND

LEDs de Control (D6-D9):
  □ LED 1 en D6 (ánodo a D6 a través de 220Ω, cátodo a GND)
  □ LED 2 en D7 (ánodo a D7 a través de 220Ω, cátodo a GND)
  □ LED 3 en D8 (ánodo a D8 a través de 220Ω, cátodo a GND)
  □ LED 4 en D9 (ánodo a D9 a través de 220Ω, cátodo a GND)

Ethernet Shield:
  □ Shield colocado correctamente en Arduino
  □ Pines alineados correctamente
  □ SD Card insertada en ranura
  □ Cable Ethernet conectado


CARGA DE CÓDIGO
═══════════════════════════════════════════════════════════════════════════════

Arduino IDE:
  □ Abrir Arduino IDE
  □ Archivo → Abrir → PanelSolarsimple.ino
  □ Seleccionar Board: Arduino Uno
  □ Seleccionar Puerto COM correcto
  □ Compilar (Sketch → Verify)
  □ Verificar sin errores
  □ Subir (Sketch → Upload)
  □ Ver mensaje "Done uploading"

Serial Monitor:
  □ Tools → Serial Monitor
  □ Seleccionar 9600 baud
  □ Presionar Reset en Arduino
  □ Ver mensajes:
    - "Inicializando SD card..."
    - "OK - SD inicializada"
    - "OK - Archivo index.htm encontrado"
    - "Servidor en: 192.168.1.177"


PREPARACIÓN SD CARD
═══════════════════════════════════════════════════════════════════════════════

Formateo:
  □ Formatear SD card en FAT16 (NO FAT32)
  □ En Windows: Click derecho → Formatear → FAT
  □ En Mac: Disk Utility → Formatear como MS-DOS

Copiar Archivo:
  □ Copiar index.htm a raíz de SD
  □ Estructura final:
     SD:/
     └── index.htm
  □ SOLO este archivo, nada más
  □ Eyectar SD card de forma segura


INSERCIÓN SD CARD EN SHIELD
═══════════════════════════════════════════════════════════════════════════════

Instalación:
  □ Desconectar Arduino de poder
  □ Tomar SD card con contactos hacia el shield
  □ Insertar en ranura con cuidado
  □ Escuchar "click" cuando encaje
  □ SD card debe quedar en su lugar
  □ NO FORZAR

Verificación:
  □ Shield debería reconocer la tarjeta
  □ Serial Monitor debería mostrar "OK - SD inicializada"
  □ Si no: verificar inserción o usar otra SD card

VALIDACIÓN SENSORES
═══════════════════════════════════════════════════════════════════════════════

Voltaje Batería (A3):
  □ Usar multímetro para medir batería 12V
  □ Comparar con valor mostrado en web (bateria_volt)
  □ Diferencia < 0.5V es aceptable
  □ Si > 1V, ajustar CALIBRACION en código

Voltaje PWM (A4):
  □ Usar multímetro para medir entrada
  □ Comparar con valor mostrado en web (pwm_volt)
  □ Diferencia < 0.5V es aceptable

LDR Sensor (A5):
  □ Cambiar iluminación sobre sensor
  □ Observar lectura ADC cambiar (0-1023)
  □ Observar intensidad % cambiar (0-100%)
  □ Valores esperados:
    - Oscuridad: 0-100
    - Luz interior: 200-500
    - Luz solar: 800-1023

CALIBRACIÓN FINAL
═══════════════════════════════════════════════════════════════════════════════

Si hay error sistemático (> 1V):

  □ Anotar voltaje con multímetro: V_real
  □ Anotar voltaje en web: V_mostrado
  □ Calcular: CALIBRACION_nueva = 0.94 * (V_real / V_mostrado)
  □ Editar PanelSolarsimple.ino línea 54
  □ Cambiar: const float CALIBRACION = 0.94;
  □ Por: const float CALIBRACION = X.XX; (valor calculado)
  □ Compilar y subir código
  □ Verificar nuevamente
  □ Repetir hasta diferencia < 0.3V
